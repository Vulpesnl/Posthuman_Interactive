    UX-UI

    Reto Sporthub Club

    Este proyecto consistió en la creación de un sistema de diseño completo para una página web destinada a una empresa que ofrece servicios de salud deportiva. El objetivo principal fue desarrollar una interfaz intuitiva y atractiva que facilitara la navegación y el acceso a los diferentes servicios ofrecidos por Sporthub Club.


    El sistema de diseño incluye una paleta de colores coherente, tipografías seleccionadas cuidadosamente, componentes reutilizables y patrones de interacción consistentes que garantizan una experiencia de usuario fluida y agradable en toda la plataforma.
